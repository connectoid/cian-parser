            <h2 class="m_ttl">Description</h2>

            <div class="ua_desc" id="desc">

                <p>HTTP Requests consist of request and header lines.  &nbsp;The initial request line includes the method, path and HTTP version.  The header lines contain fields such as Accept, Cache-Control, Host, User-Agent, etc.  &nbsp;The browser's HTTP Request asks the server to send the resource and provides the server information that could alter the language, the rendering of the page, access to the resource, etc.</p>

            </div>

            <h2 class="m_ttl">Request Line</h2>

            <div class="ua_desc" id="how" style="border-bottom: none; display: none;">

                <p>The request line is the first line of the Request header.  &nbsp;Per <a href="http://tools.ietf.org/html/rfc2616#section-5.1" target="_blank" class="link_body">RFC2616 Section 5.1</a> the request line consists of the Request-Method (GET, POST, etc) space Request-URI (path) space HTTP Version (0.9, 1.0, 1.1) CRLF.  &nbsp;No other CR's or LF's are allowed except at the end of the line.</p>

            </div>
